Projects

I have been making projects since I was 14 years old, and I don't plan on stopping. These projects were from classes, hackathons, or just done in free time on my own (like this site). In a way, this portfolio helps me see what I've learned and how far I've come as a programmer.

CryptoTrakker

Crypto-currency Rails webapp - Features login and user management system through Devise gem. Using the CoinMarketCap API, users can add and track their own cryptos as well as view their how their porfolio is doing. Front-end created using bootstrap.

Media Tone Analyzer

Enter any hashtag/keyword and our webapp collects information on the average emotion/tone associated with that topic. Uses tweepy and praw APIs to collect and python to parse data. User interface is done in Semantic UI, with Backend done in Node and ExpressJS. Developed for cuHacks 2019.

HOME

Unity game developed during Global Game Jam 2019. Enemies spawn from the top of your screen and move in random patterns all while firing at you. The goal is to defend your HOME from invaders by either using mouse or touch input to draw deflective barriers. Created using C#.

Arduino Piano

Grade 12 CS summative. Produced a musical keyboard with recording and playback functionality in Arduino. Input was done through a resistor ladder system for key inputs, and piezzo speaker for output. Included pre-programmed tunes that can be accessed through an LCD screen.

Interval Calculator

Developed an Interval calculator in C++ using classes and various data structures. The calculator would receive inputs and commands to calculate various properties, such as cartesian integer pairs and intersections. Other functions include scalar operations and memory. Created for ECE150.
